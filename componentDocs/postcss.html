    # postcss

    postcss 是一个 css 处理工具。PostCSS 的用处非常多，包括给 CSS 自动加前缀、使用下一代 CSS 语法等。

    解除了语法上的禁锢，通过插件机制来扩展语言本身.

    给 CSS 自动加前缀，增加各浏览器的兼容性：

    h1 {
    display: felx;
}

输出
h1 {
    display: -webkit-box;
    display: -webkit-flex;
    display: -ms-flexbox;
    display: flex;
}

    使用下一代 CSS 语法：

    :root {
    --red: #d33;
}
h1 {
    color: var(--red);
}

输出
h1 {
    color: '#d333'
}

    在 PostCSS 启动时，会从目录下的 postcss.config.js 文件中读取所需配置.

    # webpack 中使用

    module.exports = {
    rules: [
        {
            test: /\.css$/,
            use: ['style-loader', 'css-loader', 'postcss-loader']
        }
    ]
}


    module.exports = {
    plugins: [
        // 自动补全
        require('autoprefixer')
    ]
}
